## 💻 Sobre o projeto

O **Keyframes - Surfista Prateado** é um projeto front-end interativo criado para explorar e demonstrar o poder das animações em CSS puro utilizando a regra `@keyframes`. Com a temática do icônico personagem da Marvel, o Surfista Prateado (Silver Surfer), o site apresenta efeitos visuais imersivos e dinâmicos de forma fluida.

Este projeto foi desenvolvido como estudo prático e demonstração de habilidades em animações web e design responsivo.

🔗 **[Acesse o site rodando aqui](https://henriqtw.github.io/keyframes-surfista-prateado/)**

---

## ⚙️ Funcionalidades

- [x] Animações contínuas e imersivas utilizando CSS `@keyframes`.
- [x] Temática estilizada baseada no ambiente espacial/cósmico.
- [x] Interface limpa e focada no apelo visual.
- [x] Hospedagem direta e rápida através do GitHub Pages.

---

## 🛠 Tecnologias

As seguintes ferramentas e tecnologias foram utilizadas na construção deste projeto:

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização, layout e, principalmente, animações avançadas (`@keyframes`, `transitions`, `transforms`).
- **Git e GitHub**: Versionamento de código e hospedagem (GitHub Pages).
